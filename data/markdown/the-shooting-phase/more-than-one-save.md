# More than One Save

A model that can make both an Armour Save roll and a Ward save makes its Armour Save roll first. If this is failed (or modified to the point at which the model cannot pass it), the model makes its Ward save.

*Note that only one Ward save can be attempted and that different Ward saves cannot be combined together. If a model has more than one Ward save, simply use the best.*
