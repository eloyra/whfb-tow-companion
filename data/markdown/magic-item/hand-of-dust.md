# Hand of Dust

Enchanted Item 35 points

*The severed hand of the Great Necromancer himself, the mere proximity of this withered appendage can drain all life and vitality from mortals, leaving them as desiccated husks.*

Necromancers only. This model can cast the following [Bound spell](../magic/bound-spells.md), with a Power Level of 2. Roll a D6 each time the Hand of Dust is used. On a roll of 1, it crumbles to dust and cannot be used again:

|  |  |
| --- | --- |
| **Type** | [Assailment](../magic/assailment.md) |
| **Casting Value** | 9+ |
| **Range** | Combat |

The target enemy unit suffers 2D6 Strength 5 hits, each with an [AP](../the-shooting-phase/armour-piercing.md) of -1.
