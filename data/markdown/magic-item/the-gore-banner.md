# The Gore Banner

Magic Standard 50 points

*Enemies tremble to see the foul sigils daubed upon this blood-drenched standard.*

If a unit wishes to declare a charge against a unit carrying the Gore Banner, it must first make a [Leadership test](../model-profiles/leadership-tests.md). If this test is failed, the unit cannot charge. It does not move and is considered to have made a failed charge. If this test is passed, the unit can charge as normal.
