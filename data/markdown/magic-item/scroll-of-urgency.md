# Scroll of Urgency

Unique 30 points

*Scrolls of Urgency contain a simple but effective incantation which, when read aloud, imbues the undead servants of the Mortuary Cult with morbid vitality.*

|  |  |
| --- | --- |
| **Type** | [Conveyance](../magic/conveyance.md) |
| **Casting Value** | 7 |
| **Range** | 21" |

If the target friendly unit has already moved during this [Movement phase](https://tow.whfb.app/the-movement-phase) (not including failed charge moves), it can immediately move again.
