# Needs More Nails

Loading a cannon with shrapnel is not a precise art. When firing grapeshot, check range and declare a target as normal, then roll an Artillery dice:

- If a 'Misfire' is rolled, something has gone horribly wrong. Roll immediately on the appropriate Misfire table to determine exactly what.
- If a number is rolled, the target unit suffers that many automatic hits.
