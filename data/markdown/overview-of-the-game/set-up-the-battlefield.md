# Set up the Battlefield

Next, you will need a battlefield, which can be set up on any flat surface, be it the kitchen table, the floor or a dedicated wargames table. The players set up terrain for their armies to fight over, representing woods, fortified watchtowers and other features that make up the landscape of the Warhammer world. How to do this is covered in the [Warhammer Battles](https://tow.whfb.app/warhammer-battles) section.
