# Devolve

Signature Spell

*Delving into their enemies' minds, the Shaman magnifies the savage and animalistic parts of their psyche until they are no more than growling beasts.*

|  |  |
| --- | --- |
| **Type** | [Magic Missile](../magic/magic-missiles.md) |
| **Casting Value** | 8+ |
| **Range** | 15" |

The target enemy unit must immediately make a [Leadership test](../model-profiles/leadership-tests.md). If this test is failed, it loses a number of Wounds equal to the amount by which it failed the test.
