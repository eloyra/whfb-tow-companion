# Light of Death

Bound Spell

*The unleashed spirits plunge through the bodies of the Tomb Kings' enemies, their hapless victims suffering unbearable agony as their life essence is utterly drained.*

|  |  |
| --- | --- |
| **Type** | [Magic Missile](../magic/magic-missiles.md) |
| **Casting Value** | 7+ |
| **Range** | 36" |

The target enemy unit must immediately make a [Leadership test](../model-profiles/leadership-tests.md). If this test is passed, it suffers D3 Strength 3 hits, each with an [AP](../the-shooting-phase/armour-piercing.md) of -1. If, however, this test is failed, it suffers D6+3 Strength 6 hits, each with the [Multiple Wounds (2)](../special-rules/multiple-wounds.md) special rule and with no [armour](../the-combat-phase/make-armour-saves-combat.md) or [Regeneration](../special-rules/regeneration.md) saves permitted ([Ward saves](../the-shooting-phase/ward-saves.md) can be attempted as normal).
