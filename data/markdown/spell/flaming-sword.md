# Flaming Sword

Spell Level 1

*With a simple gesture, the Elementalist summons a great, flaming sword with which to strike at their enemies.*

|  |  |
| --- | --- |
| **Type** | [Assailment](../magic/assailment.md) |
| **Casting Value** | 7+ |
| **Range** | Combat |

A single enemy unit the caster is engaged in combat with suffers D6+1 Strength 3 hits, each with an [AP](../the-shooting-phase/armour-piercing.md) of -. These hits have the [Flaming Attacks](../special-rules/flaming-attacks.md) special rule.
