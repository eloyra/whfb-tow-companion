# Ancestral Warriors

Signature Spell

*TK*

|  |  |
| --- | --- |
| **Type** | [Assailment](../magic/assailment.md) |
| **Casting Value** | 7+/11+ |
| **Range** | Combat |

If this spell is cast with a casting result of 7 or more, a single enemy unit the caster is engaged in combat with suffers 2D3 Strength 2 hits, each with an [AP](../the-shooting-phase/armour-piercing.md) of -. If this spell is cast with a casting result of 11 or more, the target enemy unit suffers 2D6 Strength 4 hits, each with an AP of -1. This spell has the [Armour Bane (2)](../special-rules/armour-bane.md) special rule.
