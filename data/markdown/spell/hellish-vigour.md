# Hellish Vigour

Signature Spell

*The caster invigorates their ghastly warriors, who advance upon the foe with uncanny vigour.*

|  |  |
| --- | --- |
| **Type** | [Enchantment](../magic/enchantment.md) |
| **Casting Value** | 7+/10+ |
| **Range** | [Self](../magic/range-self-spells.md) |

If this spell is cast with a casting result of 7 or more, a single friendly unit that has the [Necromantic Undead](../special-rules/necromantic-undead.md) special rule and is within the caster's [Command range](../characters/command-range.md) gains the [Reserve Move](../special-rules/reserve-move.md) special rule. If this spell is cast with a casting result of 10 or more, every friendly unit that has the Necromantic Undead special rule and is within the caster's Command range gains the Reserve Move special rule. This spell lasts until the end of this turn.
