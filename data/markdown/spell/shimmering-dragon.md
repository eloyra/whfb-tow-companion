# Shimmering Dragon

Spell Level 2

*An incorporeal Dragon materialises over the battlefield, stooping low to bear aloft a mighty champion.*

|  |  |
| --- | --- |
| **Type** | [Conveyance](../magic/conveyance.md) |
| **Casting Value** | 8+ |
| **Range** | 12" |

This spell can only target friendly characters. If the target friendly character is not fleeing and has not already moved during this [Movement phase](https://tow.whfb.app/the-movement-phase), it gains the [Fly (10)](../special-rules/fly.md) special rule until the end of this turn.
