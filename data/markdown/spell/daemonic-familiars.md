# Daemonic Familiars

Spell Level 3

*The Daemonologist's many familiars fall upon the foe, slashing and biting with incorporeal claws and teeth.*

|  |  |
| --- | --- |
| **Type** | [Assailment](../magic/assailment.md) |
| **Casting Value** | 8+ |
| **Range** | Combat |

A single enemy unit the caster is engaged in combat with suffers 2D6 Strength 2 hits, with no armour save permitted ([Ward](../the-shooting-phase/ward-saves.md) and [Regeneration](../special-rules/regeneration.md) saves can be attempted as normal).
