# Vindictive Glare

Spell Level 1

*Their brows furrowed and their teeth clenched, the Shaman's glare is so intense that their enemies visibly wither.*

|  |  |
| --- | --- |
| **Type** | [Magic Missile](../magic/magic-missiles.md) |
| **Casting Value** | 8+ |
| **Range** | 21" |

The target enemy unit suffers a single Strength 7 hit with the [Multiple Wounds (D3)](../special-rules/multiple-wounds.md) special rule and with no armour save permitted ([Ward](../the-shooting-phase/ward-saves.md) and [Regeneration](../special-rules/regeneration.md) saves can be attempted as normal). This spell may target an enemy unit engaged in combat.
