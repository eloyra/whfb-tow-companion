# Blackened Bolts

Spell Level 1

*With a gesture, the Sorcerer turns the tendrils of shadow that crawl about them into bolts of dark lightning.*

|  |  |
| --- | --- |
| **Type** | [Magic Missile](../magic/magic-missiles.md) |
| **Casting Value** | 8+ |
| **Range** | 18" |

The target enemy unit suffers 2D6 Strength 3 hits, each with an [AP](../the-shooting-phase/armour-piercing.md) of -1. Any unit that suffers one or more unsaved wounds from this spell must make a [Panic test](../the-psychology-of-war/panic-tests.md) as if it had taken heavy casualties.
