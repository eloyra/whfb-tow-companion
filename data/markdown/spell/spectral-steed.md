# Spectral Steed

Spell Level 5

*As the Necromancer chants words of dark power, a ghostly steed forms from the night air.*

|  |  |
| --- | --- |
| **Type** | [Conveyance](../magic/conveyance.md) |
| **Casting Value** | 9+ |
| **Range** | 12" |

[Remains in Play](../magic/remains-in-play.md). This spell can only target friendly characters whose troop type is [infantry](../troop-types-in-detail/infantry.md). Whilst this spell is in play, the target friendly character gains the [Ethereal](../special-rules/ethereal.md) and [Fly (10)](../special-rules/fly.md) special rules.
