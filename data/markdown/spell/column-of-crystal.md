# Column of Crystal

Spell Level 3

*The Illusionist conjures forth an illusory column of crystal. All that see it believe it to be utterly impenetrable.*

|  |  |
| --- | --- |
| **Type** | [Magical Vortex](../magic/magical-vortex.md) |
| **Casting Value** | 10+ |
| **Range** | 9" |

[Remains in Play](../magic/remains-in-play.md). Place a large (5") blast template so that its central hole is within 9" of the caster. Whilst in play, the template does not move and is treated as impassable terrain over which no line of sight can be drawn.
