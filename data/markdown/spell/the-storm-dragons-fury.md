# The Storm Dragon's Fury

Signature Spell

*With a quick gesture, Miao Ying summons forth lightning from the skies to strike at her foes.*

|  |  |
| --- | --- |
| **Type** | [Magic Missile](../magic/magic-missiles.md) |
| **Casting Value** | 10+ |
| **Range** | 18" |

The target enemy unit suffers 2D3 Strength 5 hits, each with an AP of -3. These hits have the [Flaming Attacks](../special-rules/flaming-attacks.md) special rule. Once these hits have been resolved, the target enemy unit must [Give Ground](../movement-in-detail/give-ground.md).
