# Big Smartz

Signature Spell

*Gesturing lazily with a crooked staff, the caster enhances and sharpens the minds of those nearby, giving them a clearer look upon the world.*

|  |  |
| --- | --- |
| **Type** | [Enchantment](../magic/enchantment.md) |
| **Casting Value** | 8+ |
| **Range** | [Self](../magic/range-self-spells.md) |

[Remains in Play](../magic/remains-in-play.md). Whilst this spell is in play, friendly units within the [Command range](../characters/command-range.md) of this model may re-roll any failed [Stupidity](../special-rules/stupidity.md) tests. Additionally, if a friendly unit within the Command range of this model when this spell is cast failed its Stupidity test during the [Start of Turn](../the-strategy-phase/start-of-turn.md) sub-phase of the same turn, it may immediately make this test again.
