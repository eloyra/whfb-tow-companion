# Stream of Corruption

Spell Level 2

*From the fingertips of the Dark Mage corrupted magic flows, flaying the flesh of the enemy.*

|  |  |
| --- | --- |
| **Type** | [Assailment](../magic/assailment.md) |
| **Casting Value** | 8+ |
| **Range** | Combat |

Place a flame template so that the narrow end touches the caster's base edge and the broad end is over a unit they are engaged in combat with. Any model (friend or foe) whose base lies underneath the [template](../general-principles/templates.md) risks being hit and suffering a single Strength 3 hit with an [AP](../the-shooting-phase/armour-piercing.md) of -1.
