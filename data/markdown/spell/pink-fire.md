# Pink Fire

Signature Spell

*As the cackling Daemon prances and gestures, pink flames rise from the earth to envelope its enemies.*

|  |  |
| --- | --- |
| **Type** | [Magic Missile](../magic/magic-missiles.md) |
| **Casting Value** | 8+ |
| **Range** | 18" |

[Daemons of Tzeentch](../special-rules/daemon-of-tzeentch.md) only. The target enemy unit suffers D3+3 Strength 3 hits, each with an [AP](../the-shooting-phase/armour-piercing.md) of - and with the [Flaming Attacks](../special-rules/flaming-attacks.md) special rule.
