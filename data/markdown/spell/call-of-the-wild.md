# Call of the Wild

Spell Level 1

*Filled with primal energy, the Children of Chaos move across the battlefield with unwholesome vigour.*

|  |  |
| --- | --- |
| **Type** | [Conveyance](../magic/conveyance.md) |
| **Casting Value** | 8+ |
| **Range** | 12" |

This spell can only target friendly models whose troop type is [infantry](../troop-types-in-detail/infantry.md) or [cavalry](../troop-types-in-detail/cavalry.md). Until the end of this turn, the target friendly unit gains the [Reserve Move](../special-rules/reserve-move.md) special rule.
