# Burning Gaze

Signature Spell

*Beams of cleansing white light flow from the Damsel's eyes, searing all traces of evil from her sight.*

|  |  |
| --- | --- |
| **Type** | [Magic Missile](../magic/magic-missiles.md) |
| **Casting Value** | 9+ |
| **Range** | 5D6" |

Draw a straight line, 5D6" in length, from the caster's base edge. Any enemy model whose base falls under this line suffers a single Strength 4 hit with no [armour save](../the-combat-phase/make-armour-saves-combat.md) permitted ([Ward](../the-shooting-phase/ward-saves.md) and [Regeneration](../special-rules/regeneration.md) saves can be attempted as normal).
