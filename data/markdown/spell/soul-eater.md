# Soul Eater

Spell Level 6

*The Dark Mage reaches out a taloned hand to tear at the very soul of their enemy.*

|  |  |
| --- | --- |
| **Type** | [Assailment](../magic/assailment.md) |
| **Casting Value** | 7+ |
| **Range** | Combat |

A single enemy model the caster is engaged in combat with suffers a single Strength 3 hit with the [Multiple Wounds (3)](../special-rules/multiple-wounds.md) special rule and with no armour save permitted ([Ward](../the-shooting-phase/ward-saves.md) and [Regeneration](../special-rules/regeneration.md) saves can be attempted as normal).
