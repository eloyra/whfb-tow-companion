# Unquiet Spirits

Spell Level 2

*Evil spirits gather around the enemy, their spectral talons creeping beneath armour to tear at flesh.*

|  |  |
| --- | --- |
| **Type** | [Magic Missile](../magic/magic-missiles.md) |
| **Casting Value** | 8+ |
| **Range** | 15" |

The target enemy unit suffers 3D6 Strength 2 hits, with no armour save permitted ([Ward](../the-shooting-phase/ward-saves.md) and [Regeneration](../special-rules/regeneration.md) saves can be attempted as normal).
