# Steed of Shadows

Spell Level 1

*Coal-black daemonic steeds materialise from the air, bearing warriors aloft on wings of shadow.*

|  |  |
| --- | --- |
| **Type** | [Conveyance](../magic/conveyance.md) |
| **Casting Value** | 8+ |
| **Range** | 15" |

This spell can only target friendly models whose troop type is [infantry](../troop-types-in-detail/infantry.md). If the target friendly unit is not fleeing and has not already moved during this Movement phase, it gains the [Fly (12)](../special-rules/fly.md) special rule until your next [Start of Turn](../the-strategy-phase/start-of-turn.md) sub-phase.
