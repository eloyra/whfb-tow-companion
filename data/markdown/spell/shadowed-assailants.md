# Shadowed Assailants

Spell Level 4

*At the Sorcerer's command, the enemy's own shadows turn upon them, clawing at flesh with talons of darkness.*

|  |  |
| --- | --- |
| **Type** | [Assailment](../magic/assailment.md) |
| **Casting Value** | 8+ |
| **Range** | Combat |

A single enemy unit the caster is engaged in combat with suffers 3D6 Strength 1 hits, with no [armour](../the-combat-phase/make-armour-saves-combat.md) or [Regeneration](../special-rules/regeneration.md) save permitted ([Ward saves](../the-shooting-phase/ward-saves.md) can be attempted as normal).
