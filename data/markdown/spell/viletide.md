# Viletide

Signature Spell

*The Shaman calls to the creeping things that nest in the decaying undergrowth, creating a vile wave of spiders, centipedes and slugbeetles that swarm over the foe.*

|  |  |
| --- | --- |
| **Type** | [Magic Missile](../magic/magic-missiles.md) |
| **Casting Value** | 9+ |
| **Range** | 15" |

The target enemy unit suffers 5D6 Strength 1 hits, with no armour save permitted ([Ward](../the-shooting-phase/ward-saves.md) and [Regeneration](../special-rules/regeneration.md) saves can be attempted as normal).
