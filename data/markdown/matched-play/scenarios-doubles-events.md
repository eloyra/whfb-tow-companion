# Scenarios (Doubles Events)

At a Doubles event, organisers should default to using the Doubles scenarios. If the event runs for more than three rounds, organisers are free either to repeat any of the Doubles scenarios, perhaps with different [Secondary objectives](secondary-objectives.md), or to use any of the standard matched play scenarios.
