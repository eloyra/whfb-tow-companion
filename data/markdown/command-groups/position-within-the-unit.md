# Position Within the Unit

Except in rare cases, a command group will be found in the forefront of the fighting, leading by example. Unless noted otherwise, the models that make up a unit's command group must be placed in the front rank of the unit, as close to the centre of the rank as possible. If the command group includes a standard bearer, this model should take precedence in the centre of the rank (or as close to the centre as is possible), with other members of the command group placed to either side.
