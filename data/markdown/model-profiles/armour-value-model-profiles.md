# Armour Value (Model Profiles)

See also: [Armour Value](../weapons-of-war/armour-value.md)

Most models wear armour, and the type of armour they wear determines their [armour value](../weapons-of-war/armour-value.md). However, in the case of large models with multiple riders or crew, such as monsters and chariots, an armour value is not always easy to determine. To avoid confusion, and if appropriate, such models will have an armour value given within their rules.
