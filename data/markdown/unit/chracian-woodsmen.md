**Cost:** 12 points per model

**Unit Category:** [Infantry](../troop-types-in-detail/infantry.md)

**Troop Type:** [Regular Infantry](../troop-types-in-detail/regular-infantry.md)

**Base Size:** 25 x 25 mm

**Unit Size:** 5+

[Hand weapons](../weapons-of-war/hand-weapon.md), [Chracian great blades](../weapons-of-war/chracian-great-blade.md) and [light armour](../weapons-of-war/light-armour.md)

- The entire unit may:

  - Take [warbows](../weapons-of-war/warbow.md) (+1 point per model)
  - Replace [light armour](../weapons-of-war/light-armour.md) with [heavy armour](../weapons-of-war/heavy-armour.md) (+1 point per model)
- Any unit may:

  - Upgrade one model to a Chracian Captain (champion) (+7 points per unit)
  - Upgrade one model to a standard bearer (+7 points per unit)
  - Upgrade one model to a musician (+7 points per unit)
  - Have the [Lion Cloak](../special-rules/lion-cloak.md) special rule (+1 point per model)
- 0-1 unit may replace the Vanguard special rule with:

  - [Scouts](../special-rules/scouts.md) (Free)
  - [Ambushers](../special-rules/ambushers.md) (+1 point per model)
