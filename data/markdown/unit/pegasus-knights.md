**Cost:** 59 points per model

**Unit Category:** [Cavalry](../troop-types-in-detail/cavalry.md)

**Troop Type:** [Monstrous Cavalry](../troop-types-in-detail/monstrous-cavalry.md)

**Base Size:** 40 x 60 mm

**Unit Size:** 3+

- **Pegasus Knight** **:** [Hand weapons](../weapons-of-war/hand-weapon.md), [lances](../weapons-of-war/lance.md), [heavy armour](../weapons-of-war/heavy-armour.md) and [shields](../weapons-of-war/shield.md)

- Any unit may:

  - Upgrade one model to a First Knight (champion) (+7 points per unit)
  - Upgrade one model to a standard bearer (+7 points per unit)
  - Upgrade one model to a musician (+7 points per unit)
  - Purchase a magic standard worth up to 50 points
