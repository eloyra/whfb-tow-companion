# Knights Errant

| Gallant | - | 3 | 2 | 3 | 3 | 1 | 3 | 2 | 7 |
| Bretonnian Warhorse | 8 | 3 | - | 3 | - | - | 3 | 1 | - |

**Cost:** 19 points per model

**Unit Category:** [Cavalry](../troop-types-in-detail/cavalry.md)

**Troop Type:** [Heavy Cavalry](../troop-types-in-detail/heavy-cavalry.md)

**Base Size:** 30 x 60 mm

**Unit Size:** 5+

- **Knights Errant:** [Hand weapons](../weapons-of-war/hand-weapon.md), [lances](../weapons-of-war/lance.md), [heavy armour](../weapons-of-war/heavy-armour.md) and [shields](../weapons-of-war/shield.md)
- **Bretonnian Warhorse:** Iron-shod hooves (counts as a [hand weapon](../weapons-of-war/hand-weapon.md)) and [barding](../weapons-of-war/barding.md)

- Any unit may:

  - Upgrade one model to a Gallant (champion) (+6 points per unit)
  - Upgrade one model to a standard bearer (+6 points per unit)
  - Upgrade one model to a musician (+6 points per unit)
  - Purchase a magic standard worth up to 25 points
