# How Much Terrain?

If there is too much terrain, units will struggle to manoeuvre. If there is too little, the battlefield will be sparse and uninteresting. The best guide is to use one terrain feature per 12" of longest table edge (rounded up to the nearest 12"). For example, if the longest table edge is 72", six features should suffice.
