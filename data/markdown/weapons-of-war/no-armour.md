# No Armour

A model that wears no armour is considered to have an armour value of 7+ for the purposes of Armour Save roll modifiers and any rules that improve armour value. This represents the model's natural resilience and the protection proffered by the clothing and jewellery it wears, however little this may be. For example, if a model with no armour carries a shield, it would have an armour value of 6+.
