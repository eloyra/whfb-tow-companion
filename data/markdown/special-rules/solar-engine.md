# Solar Engine

A Bastiladon equipped with a Solar Engine gains the following special rules: [Solar Radiance](solar-radiance.md) & [Beam of Chotec](beam-of-chotec.md)

Whilst within 6" of one or more Solar Engines, friendly units with the [Cold Blooded](cold-blooded.md) special rule have a +1 modifier to their Initiative characteristic.

This model can cast the following [Bound spell](../magic/bound-spells.md), with a Power Level of 2:

|  |  |
| --- | --- |
| **Type** | [Magic Missile](../magic/magic-missiles.md) |
| **Casting Value** | 9+ |
| **Range** | 24" |

The target enemy unit suffers 3D3 Strength 5 hits, each with an [AP](../the-shooting-phase/armour-piercing.md) of -2. This Bound spell has the [Flaming Attacks](flaming-attacks.md) special rule.
