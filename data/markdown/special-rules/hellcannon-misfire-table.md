# Hellcannon Misfire Table

|  |  |
| --- | --- |
| **D6** | **Result** |
| **1** | **Free at Last!** The Daemon breaks loose. Every unit (friend or foe) within 3D6" suffers D6 Strength 5 hits, each with an [AP](../the-shooting-phase/armour-piercing.md) of -1. Once these hits are resolved, the Hellcannon is removed from play as a casualty. |
| **2-4** | **Chomp!** The Hellcannon's handlers are sucked towards the furnace of the ravenous Daemon. Remove one of the Hellcannon's Chaos Dwarf Handlers. |
| **5-6** | **Blooood!** The Hellcannon breaks its chains, immediately moving 3D6" as if it were subject to the [Random Movement](random-movement.md) special rule and as if this were the [Compulsory Moves](../the-movement-phase/compulsory-moves.md) sub-phase. |
