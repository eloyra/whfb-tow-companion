# Noble Disdain

A model with this special rule whose troop type is [Cavalry](../troop-types-in-detail/cavalry.md) does not have to make a [Panic test](../the-psychology-of-war/panic-tests.md) when a friendly unit whose troop type is [infantry](../troop-types-in-detail/infantry.md) is destroyed whilst within 6" of it, or when it is fled through by a friendly unit whose troop type is infantry.
