# Burning Alignment

This model can cast the following [Bound spell](../magic/bound-spells.md), with a Power Level of 3:

|  |  |
| --- | --- |
| **Type** | [Magic Missile](../magic/magic-missiles.md) |
| **Casting Value** | 8+ |
| **Range** | 15" |

The target enemy unit suffers 2D6 Strength 4 hits, each with an [AP](../the-shooting-phase/armour-piercing.md) of -. This Bound spell has the [Flaming Attacks](flaming-attacks.md) special rule.
