# Storm Call

*The oldest Dragon Ogres are able to summon storms and call lightning down from the heavens.*

This model can cast the following [Bound spell](../magic/bound-spells.md), with a Power Level of 1. This model can cast this Bound spell even if it is engaged in combat:

|  |  |
| --- | --- |
| **Type** | [Magic Missile](../magic/magic-missiles.md) |
| **Casting Value** | 7+ |
| **Range** | [Self](../magic/range-self-spells.md) |

If this Bound spell is cast, all units within 6" of this model (friend or foe), including units engaged in combat and this model, suffer D3 Strength 4 hits, each with an [AP](../the-shooting-phase/armour-piercing.md) of -1.
