# Entrenchment

*An Engineer will direct the rapid construction of effective entrenchments that protect both war machine and crew.*

During deployment, you may 'Entrench' a single non-character model whose troop type is 'war machine' for each character in your army with this special rule. An Entrenched war machine is considered to be behind partial cover and to be defending a [low linear obstacle](../battlefield-terrain/low-linear-obstacles.md). Should the war machine move for any reason, it is no longer Entrenched.
