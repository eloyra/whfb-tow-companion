# Peasantry

*The low-born peasants of Bretonnia habitually look to their knightly betters for leadership.*

If a unit with this special rule is within 6" of a friendly model that has the [Knight's Vow](the-knights-vow.md), the [Questing Vow](the-questing-vow.md) or the [Grail Vow](the-grail-vow.md), and if that model is not fleeing, this unit can use that model's Leadership characteristic instead of its own. In addition, a standard carried by a unit with this special rule cannot be counted as a trophy of war. A character with this special rule can only join a unit that also has this special rule.
