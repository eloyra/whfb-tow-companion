# Monster Slayer

*Legends tell of warriors so mighty they can slay terrible monsters with but a single blow!*

If a model with this special rule rolls a natural 6 when making a roll To Wound for an attack made in combat, it has struck a 'Monster Slaying Blow'. Enemy models whose troop type is [monster](../troop-types-in-detail/monsters.md) are not permitted an [armour](../the-combat-phase/make-armour-saves-combat.md) or [Regeneration](regeneration.md) save against a Monster Slaying Blow ([Ward saves](../the-shooting-phase/ward-saves.md) can be attempted as normal). If an enemy model whose troop type is monster suffers an unsaved wound from a Monster Slaying Blow, it loses all of its remaining Wounds.

*Note that if an attack wounds automatically, this special rule cannot be used.*
