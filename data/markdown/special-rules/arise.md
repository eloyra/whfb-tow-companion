# Arise!

*As the Liche Priest intones words of power learned long ago from dusty papyrus, the broken forms of fallen skeletal legionaries begin to rise jerkily from the dust.*

During the [Command](../the-strategy-phase/command.md) sub-phase of their turn, if they are not engaged in combat, this character may attempt to [resurrect the fallen](resurrecting-the-fallen.md) by making a [Leadership test](../model-profiles/leadership-tests.md) (using their own Leadership). If this test is passed, a single friendly unit that has the [Nehekharan Undead](nehekharan-undead.md) special rule and is within 12" of this character recovers a number of lost Wounds. However, magically repairing gigantic undead constructs is much harder than raising skeletons from the sand. Therefore, how many Wounds are recovered depends upon the unit's troop type and this character's [Level of Wizardry](../magic/levels-of-wizardry.md):

- If the unit's troop type is [regular infantry](../troop-types-in-detail/regular-infantry.md), [heavy infantry](../troop-types-in-detail/heavy-infantry.md) or [swarms](../troop-types-in-detail/swarms.md), it recovers a number of Wounds equal to this character's Level of Wizardry +D3.
- If the unit's troop type is [light cavalry](../troop-types-in-detail/light-cavalry.md), [heavy cavalry](../troop-types-in-detail/heavy-cavalry.md) or [war beasts](../troop-types-in-detail/war-beasts.md), it recovers a number of Wounds equal to this character's Level of Wizardry +1.
- If the unit's troop type is [monstrous infantry](../troop-types-in-detail/monstrous-infantry.md), [monstrous cavalry](../troop-types-in-detail/monstrous-cavalry.md) or [light chariot](../troop-types-in-detail/light-chariots.md), it recovers a number of Wounds equal to this character's Level of Wizardry.
- If the unit's troop type is [heavy chariot](../troop-types-in-detail/heavy-chariots.md), [monstrous creature](../troop-types-in-detail/monstrous-creatures.md), [behemoth](../troop-types-in-detail/behemoths.md) or [war machine](../troop-types-in-detail/war-machines.md), it recovers a single Wound.
