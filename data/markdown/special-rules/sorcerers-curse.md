# Sorcerer's Curse

*The flesh of a Sorcerer of Hashut hardens and blackens over time, eventually turning to stone.*

If this model [miscasts](../magic/miscast.md) a spell, it must immediately make a Toughness test. If this test is failed, it loses a single Wound and gains a +1 modifier to its Toughness characteristic instead of rolling on the [Miscast table](../magic/miscast-table.md). If this test is passed, it rolls on the Miscast table as normal.
