# Feed

Nominate a single enemy model in base contact with the Hell Pit Abomination to be the target of this attack. If that model is hit, it suffers D3 automatic wounds with no [armour save](../the-combat-phase/make-armour-saves-combat.md) permitted ([Ward](../the-shooting-phase/ward-saves.md) and [Regeneration](regeneration.md) saves can be attempted as normal).
