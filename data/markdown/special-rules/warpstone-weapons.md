# Warpstone Weapons

*Skaven often use the highly unstable properties of warpstone to create devilishly deadly weapons.*

A hand weapon carried by a model with this special rule has the [Magical Attacks](magical-attacks.md) special rule and an [Armour Piercing](../the-shooting-phase/armour-piercing.md) characteristic of -1.

*Note that this special rule only applies to a single, ordinary*  [hand weapon](../weapons-of-war/hand-weapon.md) *. If the model is using two hand weapons or any other sort of weapon, this special rule ceases to apply.*
