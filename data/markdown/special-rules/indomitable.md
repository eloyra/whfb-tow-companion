# Indomitable (X)

*Within the ranks of undead march creatures with an indomitable will to hang on to the last vestiges of life no matter what.*

A unit with this special rule reduces the number of wounds suffered due to the [Unstable](unstable.md) special rule by the number shown in brackets (shown here as 'X').

*Note that this special rule is not cumulative. If two or more models in a unit have this special rule, use the highest value for the entire unit. For example, if a character with Indomitable (2) joins a unit with Indomitable (1), the whole unit uses the character's Indomitable (2) special rule.*
