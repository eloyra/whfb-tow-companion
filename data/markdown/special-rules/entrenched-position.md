# Entrenched Position

*Nuln's engineers like to see their war machines well protected by earthen ramparts.*

During deployment, you may 'Entrench' a single non-character model whose troop type is [war machine](../troop-types-in-detail/war-machines.md) for each Empire Engineer your army includes. An Entrenched war machine is considered to be behind [partial cover](../the-shooting-phase/target-behind-partial-cover.md) and to be defending a [low linear obstacle](../battlefield-terrain/low-linear-obstacles.md). Should the war machine move for any reason (including making a [Vanguard](vanguard.md) move), it is no longer Entrenched.
