# Remove Casualties (Combat)

Each unsaved wound is applied to the target unit, causing one Wound to be [lost](https://tow.whfb.app/removing-casualties). When a model is reduced to zero Wounds, it becomes a casualty and is removed from play. This continues until there are no more unsaved wounds to be applied to the unit, or there are no models remaining to be removed as casualties.
