# High Linear Obstacles (Combat)

See also: [High Linear Obstacles](../battlefield-terrain/high-linear-obstacles.md)

For the purposes of combat, high linear obstacles (i.e., 2" high or more) are treated as [impassable terrain](../movement-in-detail/impassable-terrain-movement.md).
