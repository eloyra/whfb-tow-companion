# Pursuit into a Fleeing Enemy

If a pursuing unit makes contact with a fleeing enemy unit, that fleeing enemy unit is run down exactly as if [caught by a charging unit](../movement-in-detail/charging-a-fleeing-enemy.md). The pursuing unit may then attempt to reform.
